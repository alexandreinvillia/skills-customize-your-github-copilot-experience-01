# 📘 Assignment: Estruturas de Dados em Python

## 🎯 Objective

Praticar o uso de listas, tuplas, conjuntos e dicionários em Python para organizar, consultar e transformar dados de forma eficiente.

## 📝 Tasks

### 🛠️ Listas e Tuplas na Prática

#### Description
Implemente funções para manipular uma lista de notas e uma tupla com os nomes das disciplinas.

#### Requirements
Completed program should:

- Criar uma função `average_grade(grades)` que receba uma lista de números e retorne a média.
- Criar uma função `highest_grade(grades)` que retorne a maior nota da lista.
- Usar uma tupla chamada `subjects` para armazenar pelo menos 3 disciplinas e imprimir cada uma delas.

### 🛠️ Dicionários e Conjuntos

#### Description
Implemente uma função que analise uma coleção de palavras e retorne estatísticas úteis usando dicionário e conjunto.

#### Requirements
Completed program should:

- Criar uma função `word_stats(words)` que receba uma lista de palavras.
- Retornar um dicionário com as chaves `total_words`, `unique_words` e `frequencies`.
- Usar um conjunto para calcular a quantidade de palavras únicas e um dicionário para contar frequências.
